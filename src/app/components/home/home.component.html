Android Development

Kotlin is an open-source, statically-typed programming language that supports both object-oriented and functional programming. Kotlin provides similar syntax and concepts from other languages, including C#, Java, and Scala, among many others.

Kotlin has been assigned as a offical language for andorid mobile app development after Java.

Angular Development

Angular is a platform and framework for building single-page client applications using HTML and TypeScript. Angular is written in TypeScript. It implements core and optional functionality as a set of TypeScript libraries that you import into your apps.

It's a Components, Modules, Services , dependency injection, Routing along with data binding framework.

React Development

React is a declarative, efficient, and flexible JavaScript library for building user interfaces. It lets you compose complex UIs from small and isolated pieces of code called “components”.

Build encapsulated components that manage their own state, then compose them to make complex UIs.

A Stateful Component-- When a component’s state data changes, the rendered markup will be updated
